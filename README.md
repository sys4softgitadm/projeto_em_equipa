# Projeto em Equipa - Exemplo

### Introdução
Este é um exemplo de projeto em equipa (colaborativo) utilizando Git e GitHub.
O objetivo é demonstrar como múltiplos colaboradores podem trabalhar juntos num repositório Git.

### Instruções para Colaboradores
Após clonar o repositório, cada colaborador deve seguir os seguintes passos:
1. Criar um ficheiro .env a partir do ficheiro .env.example.
2. Adicionar as suas credenciais pessoais no ficheiro .env (para desenvolvimento local).

NOTA: Nunca devem preencher os seus dados reais no ficheiro .env.example

### Credenciais de Exemplo
```
MYSQL_HOST=localhost
MYSQL_USER=seu_usuario
MYSQL_PASSWORD=sua_senha
MYSQL_DB=seu_banco_de_dados
```
